BASIC URI STRUCTURE

Plugin Name

The plugin name is "baseline", all lowercase:

http[s]://<YOUR SERVER NAME>/baseline

You access the plugin functionality by placing the plugin name immediately following the server name in the REST URI.

There are no query parameters with the baseline plugin, and only GET and POST are allowed as methods.

DIRECTORIES

No Directory (Direct Call)

http[s]://example.com/baseline

Calling the plugin directly will simply list all the available plugins (including the baseline plugin).

JSON:
{
  "baseline": {
    "plugins": [
      "baseline",
      "people",
      "places",
      "things"
    ]
  }
}
XML:
<?xml version="1.0" encoding="UTF-8"?>
<baseline xmlns="http://example.com"
          xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
          xsi:schemaLocation="http://example.com http://example.com/xsd/baseline">
  <plugins>
    <value sequence_index="0">baseline</value>
    <value sequence_index="1">people</value>
    <value sequence_index="2">places</value>
    <value sequence_index="3">things</value>
  </plugins>
</baseline>

tokens

http[s]://example.com/baseline/tokens

Calling the plugin with the "tokens" directory will list all of the tokens for the current logged-in user (GET), or will create a new token (POST), if the logged-in user is a Manager User.

GET will return a list of IDs, while POST will return only one ID (the new token).

JSON (GET):
{
  "baseline": {
    "tokens": [
      9,
      2,
      3,
      4,
      5,
      6,
      7,
      8
    ]
  }
}

When a new token is created, it is added to the current login's list, and is returned in the transaction response, as a single integer.

JSON (POST):
{
  "baseline": {
    "tokens": [
      11
    ]
  }
}
XML (GET):
<?xml version="1.0" encoding="UTF-8"?>
<baseline xmlns="http://example.com"
          xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
          xsi:schemaLocation="http://example.com http://example.com/xsd/baseline">
  <tokens>
    <value sequence_index="0">9</value>
    <value sequence_index="1">2</value>
    <value sequence_index="2">3</value>
    <value sequence_index="3">4</value>
    <value sequence_index="4">5</value>
    <value sequence_index="5">6</value>
    <value sequence_index="6">7</value>
    <value sequence_index="7">8</value>
  </tokens>
</baseline>
XML (POST):
<?xml version="1.0" encoding="UTF-8"?>
<baseline xmlns="http://example.com"
          xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
          xsi:schemaLocation="http://example.com http://example.com/xsd/baseline">
  <tokens>
    <value sequence_index="0">11</value>
  </tokens>
</baseline>
